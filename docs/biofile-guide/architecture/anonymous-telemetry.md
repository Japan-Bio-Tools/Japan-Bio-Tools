# 匿名計測仕様

## この文書の位置づけ

この文書は、BioFile Guide for Structure の匿名計測、送信可能イベント、送信禁止情報、匿名化境界の詳細正本である。
親正本は [`../設計書.md`](../設計書.md) であり、この文書は親正本が定める不変条件と参照ルールに従う。

この文書は、外部メタデータAPI、判定表、UI文言、実装順序の詳細正本ではない。

---

## プライバシー前提

匿名計測は、構造本文や API 問い合わせとは独立経路で扱う。
ローカルファイル本文、座標、ファイル名、生テキスト、識別子、個別エントリを再同定できる値を計測に含めてはならない。

---

## 実行境界との関係

### 4-3. 匿名計測との境界

匿名計測は、構造本文や API 問い合わせとは独立経路で扱う。
匿名計測で送信してよいのは、イベントコードと非可逆的カテゴリのみとする。

送信禁止:

* PDB ID
* 拡張PDB ID
* ファイル名
* 入力本文
* 派生構造情報
* 個別エントリを再同定できる値

---

## 29. 匿名計測契約

送信可能:

* 画面到達イベント
* リンククリックイベント
* unknown 返却イベント
* `recommended_next_step` クリックイベント
* エラー種別コード

送信禁止:

* ローカルファイル本文
* 座標情報
* 原ファイル名
* PDB ID / 拡張PDB ID
* 個別エントリ再同定可能な情報

イベントコード例:

* `result_rendered`
* `result_unknown_rendered`
* `click_recommended_next_step`
* `click_link_rcsb`
* `click_link_pdbe`
* `click_link_pdbj`
* `click_link_molstar`
* `click_link_icn3d`
* `click_link_molmil`
* `error_parse_failed`
* `error_invalid_identifier`

---

## 禁止事項

* 匿名計測に PDB ID / 拡張PDB ID を含めない。
* 匿名計測に原ファイル名を含めない。
* 匿名計測に入力本文、座標情報、派生構造情報を含めない。
* 個別エントリを再同定できる値を含めない。
